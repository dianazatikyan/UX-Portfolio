The Wine Steward

The Wine Steward is a family owned wine shop in Pleasanton. I was contracted by ModFox Consulting and tasked with conducting user research and completing a website redesign for the Wine Steward. The redesign aimed to provide users with a clear vision of the services the Wine Steward provides, while facilitating the online shopping experience.

The Challenge

What are wine shoppers looking for?

Following 5 user interviews with frequent wine buyers and drinkers, I identified patterns and narrowed down user goals.

The research revealed many users expressed a desire to experience the atmosphere of the wine shop through the website's designs and photographs.

Product Reviews

Users wish to be able to read product reviews on the site.

Brand Reputation

Website design must reaffirm establishment and reputation.

Flavor Descriptions

Detailed descriptions of flavors help guide new wine drinkers.

Curated Photographs

Professional photographs set the tone of the brand.

User Flow

Core Actions

I prioritized three key actions which needed to be streamlined for users: purchasing wine, signing up for a newsletter, and learning about club memberships. To demonstrate the process, I created a sample user journey of an online shopper using The Wine Steward to purchase a bottle of red wine, as shown below.

Site Map

Seamless Navigation

I restructured the information architecture by card sorting existing categories into logical groups, resulting in a new sitemap. This made navigation more intuitive and minimized clicks.

Prototype

Building out the UI

My main focus during the UI development phase was to create clean, organized, and user intuitive pages.

Iterations

Does the navigation feel intuitive?

The website's main focus was the navigation bar, which underwent multiple iterations. It needed to be user-friendly and highlight the variety of products offered by The Wine Steward.

The horizontal drop-down navigation includes three main categories and two search options for wines. A/B testing revealed that users prefer menu bars with images rather than text only.

High Fidelity

Final Thoughts

During user testing of the high-fidelity prototype, efficiency was the main focus. The redesign resulted in an average of 2 clicks saved for users when searching for a specific bottle of wine, saving up to 27 seconds.
